# 競馬予想ダッシュボード

スコアリングモデルによる競馬予想・回顧ダッシュボード集です。

## ファイル命名規則

```
YYYYMMDD_[場所コード][R番号]R_[クラス]_[レース名]_pred.html   ← 予想
YYYYMMDD_[場所コード][R番号]R_[クラス]_[レース名]_review.html ← 回顧
```

**場所コード**: TK=東京 / HN=阪神 / KY=京都 / NK=中山 / CK=中京 / NG=新潟 / KK=小倉 / SP=札幌 / HK=函館 / FK=福島

**クラス**: G1 / G2 / G3 / Listed / Open / C1〜C3 / Maiden
