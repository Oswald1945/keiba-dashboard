# TARGET 配当データ エクスポート依頼仕様（馬券種ミスプライス検証用）

## 背景
馬券種の検証には各レースの**実配当（払戻）**が必要ですが、手元の結果CSVには単勝・複勝しかありません。
TARGETから下記をCSVで書き出していただければ、こちらで即バックテストします。
（自動操作を試みましたがTARGETのUIがうまく反応せず、手動書き出しが確実なためお願いです）

## 対象レース
今回のバックテスト123レースと同じ範囲：
- **期間**: 2026/02/22 〜 2026/06/14
- **場所**: 東京・京都・阪神・函館・新潟（全レースでOK。多めでも構いません、こちらで突合します）

## 欲しい列（最低限）
レース1行につき以下があれば十分です：
1. 識別子：**年月日・場所・R**（または開催回/日次/R）← こちらの race_id と突合します
2. **馬連 配当**（的中組合せと払戻金）
3. **ワイド 配当**（3組ぶん：的中組合せと払戻金）
4. **三連複 配当**（的中組合せと払戻金）
5. （あれば）**馬単・三連単 配当**
6. （あれば）1〜3着の馬番

※ TARGETのレース検索 → 集計項目メニューで「配当（払戻）」系の項目を表示に追加 → F8(出力) でCSV出力、が想定フローです。
